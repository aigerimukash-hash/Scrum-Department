# Контур — трекер корпоративных поручений

Готовый демонстрационный MVP внутреннего веб-приложения для контроля поручений, выданных на совещаниях.

## Запуск

Откройте файл `index.html` двойным кликом в любом современном браузере. Сборка, сервер и установка зависимостей не требуются.

## Доступы

- **Руководитель:** Саликов Жанболат. Введите пароль `1234` на стартовом экране. Режим открывает создание, изменение и удаление поручений.
- **Сотрудник:** войдите без пароля, указав только свою фамилию. Будут показаны поручения, где фамилия совпадает с ФИО исполнителя. Доступна лишь отправка отчёта об исполнении.

Все созданные поручения, изменения и отчёты сохраняются в LocalStorage браузера и остаются после перезагрузки страницы.

## Реализовано

- имитация бесшовного SSO-входа в один клик;
- парольный доступ для руководителя Саликова Жанболата и ограниченный вход сотрудника по фамилии;
- KPI: общие для руководителя и персональные для сотрудника;
- шесть демонстрационных поручений на русском языке, включая два просроченных;
- поиск по теме совещания и исполнителю;
- фильтры статусов;
- адаптивный реестр поручений;
- детальная панель с текстом поручения, сроком, участниками и отсутствующими;
- создание совещания и поручения руководителем, редактирование параметров и удаление;
- ручной ввод ФИО исполнителя, присутствующих и отсутствующих — без жёстко заданного справочника сотрудников;
- отправка отчёта сотрудником: статус меняется на «Выполнено», показатели пересчитываются;
- сохранение состояния в LocalStorage.

## Технологии

Ванильные HTML, CSS и JavaScript в одном файле — вариант архитектуры, предусмотренный PRD для простого развёртывания.
